# Projeto desenvolvido com o Framework .NET da Microsoft. 
### Esta aplicação foi desenvolvida na linguagem CSharp (C#) para estudo e práticas de tecnologias e aplicações em .NET.

- Criada uma solução para sortear seis números aleatórios e exibir o resultado na tela.

- Menu de apresentação: Criado para a Interface do usuário.

- Utilizada função randômica para gerar números aleatórios com exibição do resultado na tela.

- Sistema construído e testado na versão 9..0 da linguagem de programação .NET.

### Ferramentas utilizadas no projeto:

- Framework .NET da Microsoft: Utilizado para o desenvolvimento da aplicação.

- Linguagem CSharp (C#): A linguagem de programação principal utilizada.

- Função randômica: Para gerar números aleatórios.

- Visual Studio Code
